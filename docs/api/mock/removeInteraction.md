# removeInteraction

Remove interactions from the mock server. 

## Syntax

```js
removeInteraction(id)
```

## Usage

### âœ…  Correct Usage

```js

```

```js
// using multiple interaction ids

```

## Arguments

#### > id (string|string[])

Interaction ids
