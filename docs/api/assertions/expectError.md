# expectError

Assert network errors.

## Syntax

```js
expectError()
expectError(text)
expectError(error_object)
```

## Usage

### âœ…  Correct Usage

```js
// expect any error
await spec()>
  .get('/posts/5')
  .expectError();
```

```js
// expect ECONNREFUSED error text in the response
await pactum.spec()
  .get('/api/users/5')
  .expectError('ECONNREFUSED');
```

```js
// expect error object
await pactum.spec()
  .get('/api/users/5')
  .expectError({ code: 'ECONNREFUSED' });
```

## Arguments

#### > text (string)

Error text

#### > error_object (object)

Error object
